## 탬플릿 
- 탬플릿 태그에는 최상위 div 만 들어갈 수 있음.

## 디렉티브
- v-if : 조건
- v-else : 조건
- v-else-if : 조건
- v-show : 조건(css만 바꿈)
- v-bind:href : 링크
- :href : v-bind 생략
- v-bind:class : 클래스의 속성을 제어
- v-bind:style : 스타일시트 속성을 제어
- v-for : 반복루프
- v-for(val, key, index) : for 를 세분화
- 이벤트 : methods 객체와 맵핑
- 이벤트 (this)

- computed : 데이터를 바인딩하는 다른 방법
- v-model : 데이터와 양방향 바인딩
